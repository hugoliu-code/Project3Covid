Code Written by Hugo Liu, Thomas Alvarado, Constantinos Papanicolaou

This project uses a CDC covid-19 case dataset labled by county to determine how vulnerable a community was compared to another

This project uses mergesort and heapsort to sort the dataset

Data Name : United_States_COVID-19_Community_Levels_by_County_20241119
Found at: https://data.cdc.gov/Public-Health-Surveillance/United-States-COVID-19-Community-Levels-by-County/3nnm-4jni/about_data

Citation:
Center for Disease Control, United States COVID-19 Community Levels by County. Link: https://data.cdc.gov/Public-Health-Surveillance/United-States-COVID-19-Community-Levels-by-County/3nnm-4jni/about_data


HOW TO RUN THE APPLICATION ========================================================================================================


To run the backend locally:

navigate to Project3Covid/backend
- cd Backend

Run the flask application (make sure it is hosted locally at 'http:127.0.0.1:5000') (should be port 5000 by default)
- flask run

To run the frontend locally

navigate to Project3Covid/frontend/Project3Covid
- cd frontend
- cd Project3Covid

Run the next.js project
- npm run dev
- Access the application at the indicated local address (default will be http://localhost:3000)
